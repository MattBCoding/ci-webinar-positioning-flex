Position Sticky

This allows you to position something relative to the container element, until a condition is met. Then it becomes fixed. NOTE: This is all in relation to the containing element. So if you put position sticky on an element within the body, it'll be in relation to the entire page. If you put it on another element, it will become sticky or fixed, until it reaches the opposite edge of the container. Then it'll go back to relative.

Z-Index

Z-Index specifies the layer order of elements. If you think the X axis is left to right, and the Y axis is up and down. The Z axis is towards the reader and away. It controls the layer order of the elements. The higher the number the closer to the reader the element appears. Values can be positive numbers or negative. Z-Index only works on positioned elements (absolute, relative, fixed, sticky) and flex items. If two elements overlap, and they have either the same z-index value, or none specified, the elements will appear in the order they appear in the HTML
